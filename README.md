## Hyri

嗨狸，一款 Hyprland 拟态 Niri 的 dotfile

按需选择你需要的功能

## 功能

> 适用版本：
> hyprland-hidpi-xprop-git 0.55.0.r78.g864dc89
> hyprutils-git 0.13.1.r1.g3cd3972

* 截图：区域截图，屏幕截图，多屏截图，二维码扫描
* 每屏幕动态工作区切换，复刻 Niri 交互手感
* 窗口宽度切换 1.0 和原列宽（窗口宽度 - 全屏宽度）
* 拖拽窗口至两列中间时插入到新列，根据目标窗口左右 25% 的区域决定插入到窗口的左边还是右边，中间 50% 的区域保持 Hyprland 原来的逻辑
* 调整部分动画效果
* fcitx5输入法切换脚本
* Wayland - XWayland 剪贴板同步
* 一些拟态相关配置调整和自定义配置
